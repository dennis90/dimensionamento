Como calcular a bitola de fios elétricos

Antes de criar algum circuito elétrico ou mesmo alguma extensão é necessário planejar qual será a fiação necessária para evitar problemas futuros (como superaquecimento e gastos desnecessários com cabos), você pode utilizar a nossa calculadora de dimensionamento de cabos para descobrir qual a bitola do fio necessária para suas necessidades, para efetuar este cálculo, você pode utilizar os seguintes passos:

  1. Descubra qual a potência de todos os itens que serão ligados neste circuito, você pode pegar alguns exemplos de potência de alguns eletrodomésticos normalmente utilizados nesta tabela ou então somando a potência (em Watts) de todos os eletrodomésticos que serão utilizados.
  2. Informe qual a tensão da sua casa, esta informação normalmente pode ser obtida verificando a conta de energia de sua casa.

Após obter estas informações, é só entrar com os dados no formulário acima, após efetuar o cálculo, será exibida uma tabela contendo qual a dimensão adequada do fio a ser utilizado levando em consideração de quantos metros de fios serão utilizados. Esta tabela foi criada utilizando como base a norma brasileira para instalações elétricas de baixa tensão.
